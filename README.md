# STF Governance Framework

This is the repository where you can find:
* The Software Transparency (STF) Charter and Participation Agreement. You can also find these documents on the [STF website](https://www.softwaretransparency.org/membership)
* The documents, processes, policies and procedures that describes the governance of the Software Transparency Foundation Programs and Projects, where the collaboration action takes place.

## STF Governance Documents

* You can find the approved STF Charter and the Participation Agreement, targeting organizations and open source projects willing to join STF in the folder [stf-charter-and-participation-agreement](/stf-charter-and-participation-agreement).
* Contribution License Agreements can be downloaded from the folder [stf-contribution-license-agreements](/stf-contribution-license-agreements)
   * Organizations willing to participate and contribute to Software Transparency Foundation Projects must sign the Organization Contribution License Agreement (oCLA).
   * Individuals, affiliated to organizations or not, willing to participate and contribute to Software Transparency Foundation Projects must sign the Individual License Agreement (iCLA).
* You can find the main document that describes the key governance aspects of STF Programs and Projects, as well as the STF Policies, in the folder [stf-programs-and-projects-governance-documents](/stf-programs-and-projects-governance-documents)

## Contact

The best way to get help is to join the STF [main mailing list](https://www.freelists.org/list/st-foundation). You can also use the [contact form](https://www.softwaretransparency.org/contact) available on STF's website. Send a communication and the STF staff will come back to you as soon as possible.
