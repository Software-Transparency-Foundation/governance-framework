# STF Governance Framework

This is the repository where you can find:
* The Software Transparency (STF) Charter and Participation Agreement. You can also find these documents on the [STF website](https://www.softwaretransparency.org/membership)
* The documents, processes, policies and procedures that describes the governance of the Software Transparency Foundation Programs and Projects, where the collaboration action takes place.

## Contact

The best way to get help is to join the STF [main mailing list](https://www.freelists.org/list/st-foundation). You can also use the [contact form](https://www.softwaretransparency.org/contact) available on STF's website. Send a communication and the STF staff will come back to you as soon as possible.
